                               见 一 面———
              Claude Code 的优雅 macOS 原生桌面客户端
================================================================================

简介
--------------------------------------------------------------------------------
「见一面」是一个基于 SwiftUI 的 macOS 原生应用，为 Claude Code 命令行工具
提供美观、流畅的图形化交互体验。支持多对话管理、流式渲染、文件上下文、
拖拽导入等特性，让你与 AI 的每一次对话都舒适自然。



功能特性
--------------------------------------------------------------------------------
* 多对话管理       侧边栏支持创建、切换、重命名和删除对话，每个对话绑定
                   独立的工作目录

* 流式输出          AI 回复实时流式渲染，支持 Markdown（标题、代码块、
                   列表、引用、链接、行内代码等）

* 文件上下文        每个对话可选择本地文件夹作为工作目录，AI 可直接阅读
                   和编辑该目录下的文件

* 拖拽导入          支持将文件或文件夹拖入对话窗口，自动复制到项目目录

* 权限模式          四种权限模式可切换：默认（按需请求）、接受编辑、
                   计划模式、自动批准

* 主题切换          支持浅色 / 深色 / 跟随系统三种外观模式

* 会话持久化        所有对话和消息自动保存在 ~/.claude/conversations/ 下，
                   关闭应用不会丢失

* 原生体验          macOS 原生窗口风格，隐藏标题栏，支持侧边栏快捷键，
                   最小窗口尺寸 900x520


系统要求
--------------------------------------------------------------------------------
* macOS 14.0 (Sonoma) 或更高版本
* Claude Code 命令行工具已安装并可在终端中使用
  （路径：~/.local/bin/claude、/opt/homebrew/bin/claude、
   /usr/local/bin/claude、/usr/bin/claude 或 ~/.nvm/ 下）
* Xcode 15.0+（如需从源码构建）


快速开始
--------------------------------------------------------------------------------
直接使用：
  1. 打开「见一面.dmg」
  2. 将「见一面.app」拖入「Applications」文件夹
  3. 首次打开时，若 macOS 提示安全警告，请在
     「系统设置 → 隐私与安全性」中允许运行
  4. 点击左侧 + 按钮，选择一个工作目录，创建新对话
  5. 输入消息，按 Enter 发送；Shift+Enter 换行

从源码构建：
  # 生成 Xcode 项目文件
  ruby gen_xcode.rb > ClaudeChat.xcodeproj/project.pbxproj

  # 使用 xcodebuild 构建
  xcodebuild -project ClaudeChat.xcodeproj \
    -scheme 见一面 \
    -configuration Release \
    -derivedDataPath ./build

  # 或直接在 Xcode 中打开 ClaudeChat.xcodeproj 运行

打包 DMG：
  ./make_dmg.sh

  Python 3       DMG 背景图生成（Pillow）
  Claude CLI     AI 引擎后端
